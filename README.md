这是一个以格雷科技现代版为主要模组的科技类空岛包。 从一棵树、一片土开始，通过各种手段获得多种资源，一步步建立起你的工厂，自动化生产一切。  
整合包目前仍处于开发阶段，前期游戏可以进行，但是仍可能进行大的更改。  
请注意：创建世界时请选择"空岛"世界类型。
